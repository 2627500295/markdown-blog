# Atom 让 JSX 后缀支持 Emmet

## 中文
设置 > 用户键盘映射

## 英文
Preferences > Keymap

```YAML
# Auto expanding for emmet @
'atom-text-editor[data-grammar="source js jsx"]': 'tab': 'emmet:expand-abbreviation-with-tab'
```
